== Toca instalar ==
Java JDK 21: https://www.oracle.com/java/technologies/downloads/#java21
Micronaut: https://micronaut.io/download/
Dynamo local: https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/DynamoDBLocal.DownloadingAndRunning.html#DynamoDBLocal.DownloadingAndRunning.title
AWS CLI: https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html

== Comandos Dynamo ==
Arrancar Dynamo local: java -Djava.library.path=./DynamoDBLocal_lib -jar DynamoDBLocal.jar -sharedDb
Crear tabla clientes: aws dynamodb create-table --table-name clientes --attribute-definitions AttributeName=id,AttributeType=N --key-schema AttributeName=id,KeyType=HASH --billing-mode PAY_PER_REQUEST --endpoint-url http://localhost:8000
Borrar tabla clientes: aws dynamodb delete-table --table-name clientes --endpoint-url http://localhost:8000
Listar las tablas: aws dynamodb list-tables --endpoint-url http://localhost:8000

== Comandos Gradlew ==
Limpiar proyecto: gradlew clean
Detener el servicio: gradlew --stop
Iniciar el servicio: gradlew run
Generar para AWS: gradlew clean shadowJar
